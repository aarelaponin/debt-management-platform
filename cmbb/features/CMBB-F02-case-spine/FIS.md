## 3. Design decisions & assumptions
1. **A1:** `caseRef` generated by TransitionGuard from `mm_case_type.idFormat` (per-type prefix needs runtime logic — custom-ID plugin trigger). Form field readonly.
2. **A2:** OnHold is data-level (currentState + DeadlineEngine pause in S2), NOT an XPDL stop — envelope stays the only XPDL (CAD rule).
3. **A3:** CLOSE authority check stubbed behind property `requireDecision=false` until F08 delivers the decision service; recorded, revisited at F08.
4. **A4:** Userview permission: `Case work` category carries role gate; whole-userview LoggedInUserPermission set with this batch (DX9-DELTAS item).
5. **A5:** Re-open (WF-FR-005) = new process instance on same cmCase row + cmEvent(CASE_REOPENED, link to closing event); authority via mmTransition actorRoles.
6. No blockers — G2 may pass.
